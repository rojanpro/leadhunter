# Message Templates

Supported variables:

- `{{business_name}}`
- `{{category}}`
- `{{city}}`
- `{{country}}`
- `{{website_status}}`
- `{{lead_reason}}`
- `{{your_name}}`
- `{{agency_name}}`
- `{{offer}}`
- `{{unsubscribe_text}}`

Always include explicit opt-out text in WhatsApp and email templates.
